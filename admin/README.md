# Delirium — админ-панель (localhost)

Минимальная локальная панель: сброс паролей, модерация аккаунтов, просмотр крашей.
Работает через Firebase Admin SDK (обходит правила), слушает только `127.0.0.1`.

## Настройка (один раз)

1. **Ключ сервис-аккаунта.** Firebase Console → ⚙️ Project settings → вкладка
   **Service accounts** → **Generate new private key**. Сохрани файл как:
   ```
   admin/serviceAccountKey.json
   ```
   (он в `.gitignore`, в репозиторий не попадёт).

2. **Установить зависимости:**
   ```sh
   cd admin
   npm install
   ```

## Запуск

```sh
cd admin
npm start
```
Открой в браузере: **http://127.0.0.1:4000**

## Что умеет
- **Пользователи** — список всех аккаунтов; сменить пароль любому (на случай «забыл
  пароль»); удалить аккаунт (вместе со связями и фото — через Cloud Function-очистку).
- **Ошибки** — последние краши приложения (устройство, версия, стек), пришедшие из
  приложения в коллекцию `crashReports`.
